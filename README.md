# siwe.vim

Syntax highlighting of [EIP-4361: Sign-In with Ethereum (SIWE)][EIP-4361] messages in [Vim][].

## Install

```
$ git clone https://github.com/spruceid/siwe.vim ~/.vim/pack/plugins/start/siwe.vim
```
